# wechat_cloudMusicApp
# 网易云音乐-微信小程序App

## 本程序所依赖服务 https://github.com/Binaryify/NeteaseCloudMusicApi
## 本程序是由微信小程序开发的 网易云音乐小程序app 仅供于练习微信小程序基本用法

### 由于本人是第一次使用微信小程序开发,所以有很多冗余代码和对一些异步操作处理不当的地方,所以只是一个很粗糙的版本,仅供学习之用.
